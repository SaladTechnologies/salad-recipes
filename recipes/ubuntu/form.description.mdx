A basic Ubuntu 24 image run on a RTX 3090 (24GB) to get started with SaladCloud.

Includes VS Code (Optional) and JupyterLab for easy access.

[Learn how to set up VS Code over Devtunnel](https://docs.salad.com/tutorials/vscode-remote-development#create-a-tunnel-and-generate-access-tokens) and fill in your tunnel ID and token in the form below. If ommitted, you can still access JupyterLab and the web terminal, but the VS Code server will not be started.
